Cloud & Infrastructure Operations Engineer

Salary: 1,500 KWD / Month

Location: Kuwait

Notes: Resident Engineer (Onsite)

Position Overview

Responsible for the day-to-day operation, monitoring, administration, and optimization of the customer's hybrid cloud and on-premises infrastructure. Manage Microsoft Azure, Nutanix, Cohesity, Fortinet security platforms, and Microsoft Defender while ensuring system availability, security, disaster recovery readiness, and operational excellence.

Required Experience

  • Strong experience administering Microsoft Azure (IaaS, PaaS)
  • Azure RBAC, Policies, Governance and Cloud Security
  • Experience managing Nutanix AHV, AOS and Prism
  • Virtualization, storage provisioning and cluster administration
  • Cohesity backup, replication and disaster recovery
  • Fortinet Firewall administration (FortiGate)
  • Web Application Firewall (WAF)
  • Microsoft Defender (Endpoint, Identity and Cloud Apps)
  • Threat monitoring and vulnerability management
  • Networking, virtualization and cybersecurity best practices
  • L2/L3 Infrastructure support
  • Vendor coordination and incident management
  • Documentation and operational reporting

Certifications

  • Bachelor's degree in Computer Science, Computer Engineering or equivalent
  • Microsoft Azure Administrator (AZ-104)
  • Nutanix Certified Professional (NCP)
  • Fortinet NSE 4
  • Cohesity Certified Administrator (CCA)
  • Microsoft Security (SC-200)
  • ITIL v4 (Preferred)
